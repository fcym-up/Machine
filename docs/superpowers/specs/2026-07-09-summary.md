# 2026-07-09 工作汇总

## 一、采集器 v7 —— 停留时段追踪

将采集器从 PowerShell 脚本重写为 Python 单文件（`collector/collector.py`），
使用 ctypes 直接调用 Win32 API，零外部依赖。

**核心思路：** 不再按轮询频率记录"切换瞬间"，改为记录"停留时段"。
窗口切换时才发出上一条事件的停留时长。同一窗口待 30 分钟只产生一条事件，
天然消灭了重复刷屏问题。

**关键修复：**
- sender 检查被 `continue` 跳过 → sender check 移到 try/except 循环体外
- 噪音过滤器过于激进 → 只过滤 Program Manager
- 正则中 `\x00` 在 Python 源码中变成真实 null 字节导致 SyntaxError
  → 改用 `[^\x01-\x7f]`
- `app/services/event.py` 第 46 行 `IndentationError`
  → 手动修复（删除 dedup 代码时遗留的缩进问题，该 bug 导致所有 POST /events 返回 500）

**Clean-Title 正则：**
- 去除浏览器后缀：` - Microsoft.*Edge$`、` - Google Chrome$`
- 去除 Edge 多标签前缀：` \S+ \d+ [^\x01-\x7f]+$`（匹配"和另外 N 个页面"）
